撰寫 powershell

windows 提供了兩種方法給使用者進行 powershell script 的編寫：

  1. 透過文字編輯器 (notepad, notepad++)進行撰寫。
  2. 透過 powershell ISE進行撰寫。

以下是針對第一個方式的詳細說明：

使用文字編輯器

打開你的記事本 /notepad++

開始編寫你的 script，以下用 docker 啟動三個 hadoop container 並且直接進入 master 的 bash 為例子：

docker start hadoop-master
docker start hadoop-slave1
docker start hadoop-slave2
docker exec -it hadoop-master bash

寫完之後選擇路徑、存檔，副檔名記得填寫成 .ps1 (這是powershell script的檔案類型就好比 unix-like system 的 script 檔案類行為 .sh)

執行 powershell script

由於 windows 預設關閉 powershell 執行 script 的功能，因此我們得透過改變執行原則 (execution policy) 來達成目的。以下為操作方式：

  1. 以系統管理員身分執行 powershell
  2. 輸入以下指令
Set-ExecutionPolicy RemoteSigned

然後電腦會問你是否要接受變更執行原則，輸入 A (accept) 表示接受。

完成後我們便可以透過 powershell 執行我們所撰寫的 script了。

執行方式

跟 unix-like system 一樣，我們只要透過 ./yourscript.ps1 就可以執行我們的 script了。
